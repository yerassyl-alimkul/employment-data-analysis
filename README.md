# Employment Data Analysis

**Tools:** R, RStudio  
**Dataset:** Employment survey data (Italy)  
**Course:** Statistics — MSc in Data Science, University of Messina (2026)

## Objective
Analysed a large-scale employment dataset to uncover patterns in salary, gender, age, and citizenship status.

## Methods
- Descriptive statistics (mean, median, SD, skewness, CV)
- Contingency table analysis (Chi-squared, Cramér's V)
- Linear regression (least squares)
- Normality testing (95% intervals)
- Data visualisation (10+ charts)
